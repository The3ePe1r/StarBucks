## ☕ About The Project

**Starbucks Coffee Website** is a fully responsive, front‑end clone of the iconic Starbucks homepage.  
It captures the warm, modern aesthetic of the brand — showcasing seasonal drinks, promotions, and the signature green‑and‑white palette.

This project was crafted to sharpen my skills in:

- Semantic HTML5
- Modern CSS3 (Flexbox, Grid, custom properties)
- Vanilla JavaScript interactivity
- Mobile‑first responsive design

Perfect for anyone learning front‑end development or looking for a beautiful open‑source coffee shop template.

---

## ✨ Features

- **🍩 Hero Section** – Eye‑catching call‑to‑action with smooth animations.
- **📱 Fully Responsive** – Looks stunning on all devices (mobile, tablet, desktop).
- **🎨 Brand‑Accurate Colors** – Official Starbucks green, gradients, and fonts.
- **🧭 Interactive Navigation** – Fixed navbar with scroll effect and mobile hamburger menu.
- **🖼️ Dynamic Product Cards** – Hover effects and elegant overlays.
- **⚡ Fast Performance** – Zero dependencies, pure vanilla code.
- **🧪 SEO‑Friendly** – Semantic tags, meta descriptions, and Open Graph support.
- **♿ Accessible** – Proper ARIA labels, keyboard navigation, and sufficient contrast.

## 🚀 Getting Started

Follow these steps to get a local copy up and running in minutes.

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge)
- A text editor (VS Code recommended)
- Git (optional, for cloning)

### Installation

1. **Clone the repo**

   ```bash
   git clone https://github.com/The3ePe1r/Starbucks.git
   ```

2. **Navigate to the project folder**

   ```bash
   cd Starbucks
   ```

3. **Open `html/home.html` in your browser**

---

## 🖥️ Usage

Customize the content to make it your own:

- Edit `index.html` to change text, images, and links.
- Tweak `css/style.css` to play with colors, fonts, and layout.
- Add interactivity in `js/script.js`.

Need to replace placeholder images? Drop your own `.png` or `.webp` files into the `images/` folder and update the `<img>` tags accordingly.
